# Note-Taker

## User Story
    As a user, I want a simple note-taker application that I can add notes to, and then see the notes that have been saved.

## Installation
    npm i
    node server.js
    git init (Heroku)

## Usage
    When you enter the website, click the "Get Started" button to get to the /notes page. Then, the user can enter a new note, and add it to the list on the left by clicking the save ðŸ’¾ button in the top right corner of the screen.
